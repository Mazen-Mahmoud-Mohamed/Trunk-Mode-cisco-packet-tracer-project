# Cisco Packet Tracer Networking Projects

A collection of networking projects built using **Cisco Packet Tracer** to practice network design, configuration, troubleshooting, and network management concepts.

## Overview

This repository contains multiple networking labs and simulations covering essential networking topics such as VLANs, trunking, routing, switching, IP addressing, and network troubleshooting.

The projects are designed for students, networking enthusiasts, and anyone preparing for networking certifications such as **CCNA**.

---

* VLAN Configuration
* Trunk Mode Configuration
* Router and Switch Configuration

* Network Troubleshooting

## Included Projects

### VLAN Configuration Project

* Creating VLANs
* Assigning switch ports
* Testing VLAN communication

### Trunk Mode Configuration Project

* Configuring trunk links between switches
* Allowing multiple VLANs over a single connection
* Verifying trunk operation

### Router and Switch Configuration

* Basic router setup
* Interface configuration
* Static IP addressing

### Small Office Network Simulation

* End-device connectivity
* Switch-to-switch communication

## Getting Started

1. Clone the repository.
2. Open any `.pkt` file using Cisco Packet Tracer.
3. Explore the topology.
4. Review the configurations.
5. Test connectivity and troubleshoot if needed.
